# Landing Page — Instituto Baktaran Irmãs Coragem

Versão inicial criada pela Mireva Design & Sistemas.

## Escopo

- Landing page responsiva em uma única página.
- Sem login, banco de dados, painel administrativo ou sistema interno.
- Doações direcionadas para Pix e Vakinha.
- Contato direcionado para o Instagram oficial.
- Conteúdo de animais e histórias mantido genérico até o Instituto fornecer dados e imagens autorizadas.

## Como visualizar

Abra o arquivo `index.html` no navegador.

Para servir localmente:

```bash
python -m http.server 8080
```

Depois, acesse `http://localhost:8080`.

## Antes de publicar

Validar com o Instituto:

1. Textos institucionais.
2. Chave Pix e link da Vakinha.
3. Fotografias e autorização de uso.
4. Regras de adoção e apadrinhamento.
5. Canais oficiais de contato.
6. Uso do crédito da Mireva no rodapé.
7. Divulgação no portfólio e nas redes.

## Arquivos

- `index.html`: estrutura e conteúdo.
- `styles.css`: identidade visual e responsividade.
- `script.js`: menu, cópia do Pix, compartilhamento e animações.
- `assets/logo-ibic.png`: logotipo fornecido.
